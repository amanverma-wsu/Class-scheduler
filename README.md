# Class Scheduler (Free Coursicle-style Starter)

A lightweight, free class schedule planner you can run locally or host anywhere.

## Features

- Add classes with:
  - Course name
  - Days (Mon–Sun)
  - Start/end times
  - Location
  - Instructor
  - Color tags
- Weekly visual schedule grid.
- Conflict detection (overlapping classes).
- Dynamic calendar range based on your earliest/latest class times.
- Save automatically to browser local storage.
- Import/export schedule JSON.

